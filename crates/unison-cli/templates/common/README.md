# {{PROJECT_NAME}}

A Unison 2D game.

## Build

```bash
unison dev web        # web dev server
unison build all      # production builds for every enabled platform
unison doctor         # verify your toolchain
```

Open `platform/ios/{{PROJECT_NAME}}-ios.xcodeproj` in Xcode for iOS development;
open `platform/android/` in Android Studio for Android.
